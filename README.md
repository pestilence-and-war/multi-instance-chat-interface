# Multi-Instance AI Digital Office Interface (v2)

A sophisticated, web-based platform designed to manage simultaneous AI agent sessions within an extensible **Digital Office** framework.

## New in v2
-   **Office Abstraction**: Deploy entire pre-staffed departments (Software Studio, Marketing Agency) with a single command.
-   **Recursive Planning**: Integrated **Strategist** persona for automatic roadmap generation.
-   **Kanban State Machine**: Persistent milestone tracking ensures projects can be resumed or recovered after failure.
-   **Global Telemetry**: Real-time monitoring tab showing agent thoughts, text streaming, and tool execution.
-   **Anti-Fraud Hardening**: Cynical auditing protocols that reject "simulated" work and demand raw tool output.
-   **Windows-Aware Execution**: Robust shell tools with background service management and automatic command translation.

## Core Features
-   **Dynamic Tabbed Interface**: Manage multiple agents (Ollama, Google, OpenRouter) in parallel.
-   **Shared Project Journal**: A unified versioned state for all specialists.
-   **Autonomous Orchestration**: Project Managers hire, verify, and re-delegate work automatically.
-   **Vision Integration**: Specialized tools allow agents to "see" and verify image assets.
-   **Safe Shell Execution**: Run code, tests, and servers directly within the workspace sandbox.

## Quick Start
1.  **Configure**: Set your API keys in `.env` and launch the server.
2.  **Initialize**: Start a session with the **Architect**.
3.  **Staff**: Request an office (e.g., "Set up a Software Studio for my new project").
4.  **Monitor**: Open the **📡 Office Telemetry** tab to watch the office work in real-time.
